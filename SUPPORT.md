## 🐛 Reporting Bugs

When reporting bugs, please include:

1. **QBioCode version**: Run `pip show qbiocode`
2. **Python version**: Run `python --version`
3. **Operating System**: e.g., Ubuntu 20.04, macOS 12.0, Windows 10
4. **Steps to reproduce**: Clear, step-by-step instructions
5. **Expected behavior**: What you expected to happen
6. **Actual behavior**: What actually happened
7. **Error messages**: Full error traceback if applicable
8. **Code sample**: Minimal code that reproduces the issue

## 💡 Feature Requests

We welcome feature requests! When suggesting a feature:

1. **Check existing issues**: Someone might have already suggested it
2. **Describe the problem**: What problem does this solve?
3. **Propose a solution**: How would you like it to work?
4. **Provide use cases**: How would you use this feature?
5. **Consider alternatives**: What other solutions have you considered?
